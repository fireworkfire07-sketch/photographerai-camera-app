# PHOTOGRAPHERAI CAMERA APP

AI powered mobile photography assistant.

## Features

- Live camera preview
- AI pose guidance
- Voice assistant
- Auto burst capture
- Multi-frame shooting
- Hero shot engine
- Smart photo selection
- Cinematic framing
- AI export system

## Stack

- React Native
- Expo Camera
- FastAPI
- Supabase
- Cloudinary

## Goal

Turn phone camera into a professional photographer assistant.
